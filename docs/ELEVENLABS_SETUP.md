# ElevenLabs AI Voice Integration Setup

Этот гайд поможет вам настроить интеграцию ElevenLabs для голосового AI-бота в вашей игре.

## 🎯 Что было добавлено

### 1. **AI Bot с голосовым приветствием**
   - Интерактивный модальный диалог с приветствием игрока
   - Пошаговый онбординг с голосовыми подсказками
   - Анимированная визуализация голоса (waveform)

### 2. **Интерактивный демо-блок** (как на сайте ElevenLabs)
   - Поле для ввода текста
   - Выбор голоса из доступных
   - Настройки голоса (стабильность, схожесть)
   - Генерация и воспроизведение речи

### 3. **ElevenLabs API сервис**
   - Полная интеграция с ElevenLabs API
   - Кэширование аудио для оптимизации
   - Поддержка streaming и regular TTS

## 🚀 Быстрый старт

### Шаг 1: Получите API ключ

1. Зайдите на [ElevenLabs](https://elevenlabs.io/)
2. Зарегистрируйтесь или войдите в аккаунт
3. Перейдите в [Settings → API Keys](https://elevenlabs.io/app/settings/api-keys)
4. Скопируйте ваш API ключ

### Шаг 2: Настройте переменные окружения

1. Создайте файл `.env` в корне проекта:
   ```bash
   cp .env.example .env
   ```

2. Откройте `.env` и вставьте ваш API ключ:
   ```env
   VITE_ELEVENLABS_API_KEY=your_api_key_here
   ```

### Шаг 3: Запустите проект

```bash
npm run dev
```

Теперь при загрузке игры появится AI-бот с голосовым приветствием! 🎉

## 🎨 Возможности AI бота

### Онбординг
AI бот приветствует игрока и объясняет правила игры в 5 шагов:
1. Приветствие
2. Объяснение цели игры
3. Инструкция по управлению
4. Информация о бустерах
5. Призыв к действию

Каждый шаг сопровождается:
- ✅ Голосовой озвучкой через ElevenLabs
- ✅ Анимированным текстом
- ✅ Визуализацией голосовой волны
- ✅ Индикатором прогресса

### Демо-блок (тестирование голоса)
Пользователи могут:
- Ввести любой текст
- Выбрать голос из доступных
- Настроить параметры:
  - **Stability** (стабильность) - насколько предсказуемым будет голос
  - **Similarity** (схожесть) - насколько точно воспроизводится оригинальный голос
- Сгенерировать и прослушать речь

## 🔧 Настройка

### Изменить сообщения онбординга

Отредактируйте файл `src/ui/AIBotModal.js`:

```javascript
this.onboardingMessages = [
  {
    text: "Ваше приветственное сообщение",
    duration: 3000
  },
  // Добавьте больше шагов...
];
```

### Изменить голос по умолчанию

В файле `src/services/ElevenLabsService.js`:

```javascript
// По умолчанию используется "Rachel"
// Измените на любой доступный голос:
const defaultVoice = this.voices.find(v => v.name === 'Bella') || this.voices[0];
```

### Отключить AI бота

Если вы хотите временно отключить AI бота, просто не указывайте API ключ в `.env`. Игра автоматически перейдет к обычному стартовому экрану.

## 🎮 Как это работает в игре

1. **Загрузка игры** → Инициализация ElevenLabs сервиса
2. **Появление AI бота** → Автоматический запуск онбординга
3. **Пользователь может:**
   - Прослушать онбординг полностью
   - Пропустить (кнопка "Skip Intro")
   - Протестировать голоса в демо-блоке
4. **Завершение** → Переход к игре

## 🐛 Решение проблем

### AI бот не появляется
- Проверьте, что API ключ установлен в `.env`
- Откройте консоль браузера и проверьте ошибки
- Убедитесь, что вы перезапустили dev сервер после изменения `.env`

### Не воспроизводится звук
- Проверьте API ключ
- Проверьте лимиты вашего ElevenLabs аккаунта
- Откройте консоль и проверьте ошибки сети

### Долгая загрузка голосов
- Это нормально при первом запуске
- Голоса кэшируются для последующих использований
- Убедитесь в стабильном интернет-соединении
